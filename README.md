## About

Geetanjali Music Academy is a dedicated platform for managing music education — connecting students with their mentors, tracking class progress, and enabling live video sessions, all in one place.

Built for Geetanjali Music Academy, the app streamlines everything from student enrollment to lesson tracking and live video classes.

---

## Features

### For Students
- Register and get approved by the academy admin
- Connect with assigned mentors
- Join live video classes directly from the app
- View class history and session details

### For Mentors
- View all enrolled students
- Track class history per student — total classes, last class date, session status
- Start and manage live video sessions with Jitsi Meet
- Add notations and notes for students

### For Admins
- Review and approve student registrations (with selfie verification)
- Assign mentors to students
- Manage the full roster of users

---

## App Flow

```
Register → OTP Verification → Selfie Upload → Admin Approval → Home
```

Role-based access ensures students, mentors, and admins each see a tailored experience.
